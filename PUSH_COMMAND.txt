# HOW TO PUSH TO GITHUB WITH TOKEN

# Replace YOUR_TOKEN_HERE with your actual GitHub token
git remote set-url origin https://YOUR_TOKEN_HERE@github.com/YusuphSalimu/tanzania-real-estate-ai.git

# Then push
git push -u origin main

# git remote set-url origin https://ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx@github.com/YusuphSalimu/tanzania-real-estate-ai.git
